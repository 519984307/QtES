# QtES
## Qt Extension System

Qt Extension System is just a Qt application development that use plugin framework from Qt Creator plugin system.

## How to build

### 1. Use cmake and Qt4

```shell
$ mkdir build
$ cd build
$ cmake ..
$ make
```

### 2. Use .pro( Qt4 and Qt5+)

1. Open QtES.pro with Qt Creator
2. Select a Kit
3. Build & Run
